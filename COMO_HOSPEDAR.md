# 🚀 Como tornar seu site Permanente

O link fornecido pelo Manus é temporário. Para ter um site que nunca sai do ar, siga estas instruções simples para hospedar no **Vercel** (Recomendado e Gratuito).

## 1. Preparação
1. Baixe o arquivo `herois-unidos-3.0-final.tar.gz` que te enviei.
2. Extraia os arquivos em uma pasta no seu computador.

## 2. Subir para o GitHub (Opcional, mas recomendado)
1. Crie uma conta em [github.com](https://github.com).
2. Crie um novo repositório chamado `herois-unidos`.
3. Use o botão "Upload files" para subir todos os arquivos da pasta extraída.

## 3. Hospedar no Vercel (Rápido e Grátis)
1. Vá para [vercel.com](https://vercel.com) e crie uma conta (pode usar seu GitHub).
2. Clique em **"Add New"** > **"Project"**.
3. Se você subiu para o GitHub, selecione o repositório. 
   - *Se não quiser usar GitHub, você pode arrastar a pasta do projeto direto para a área de upload do Vercel.*
4. O Vercel vai identificar que é um projeto **Vite**.
5. Clique em **"Deploy"**.

## 4. Seu link permanente!
Após alguns segundos, o Vercel te dará um link como `herois-unidos.vercel.app`. Este link **nunca expira** e você pode mandar para todos os seus jogadores!

---
*Dica: Se precisar fazer mudanças no futuro, basta editar os arquivos e subir novamente.*
